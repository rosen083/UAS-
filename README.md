## 2. Deskripsi Dataset
Dataset mtcars berisi data spesifikasi mobil.
Variabel yang digunakan:
- mpg: efisiensi bahan bakar
- wt : berat mobil

## 3. Cara Menjalankan
1. Pastikan R terinstal
2. Jalankan script secara berurutan:
   - 01_deskriptif.R
   - 02_normalitas.R
   - 03_korelasi.R
   - 04_regresi.R

## 4. Hasil dan Interpretasi
- Data MPG memiliki variasi sedang
- Distribusi mendekati normal
- Terdapat korelasi negatif kuat antara berat dan MPG
- Model regresi linear cukup baik

## 5. Kesimpulan
Berat mobil berpengaruh signifikan terhadap efisiensi bahan bakar.
